    Studio Ghibli API

    I built this website using HTML, CSS, JavaScript and the Studio Ghibli API. I love the world of Studio Ghibli and wanted experience with working with API's, so the Studio Ghibli open source API was a great way to get my feet wet. I learned how to make an API call, see the API data in the console, isolate specific API data, and how to display API data on a website. This project was challenging for me because I am still learning JavaScript and had to research what 'map' did, and what the best way to call an API is. I decided to use the 'async fetch' method. Overall this was a fun little project that taught me a lot about how API's work, and the JavaScript programming language.
